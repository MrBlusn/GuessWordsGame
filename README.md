# GuessWordsGame

     这是一个猜单词游戏，好早好早之前做的

    游戏规则为，给10次机会猜单词中的字母，直至完全猜出整个单词，满分10分，猜错几次扣几分，若机会用完后还未猜出单词，则会揭示单词
